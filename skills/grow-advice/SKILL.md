# 🌱 Grow Advice Skill

## Purpose
Provide indoor/outdoor cannabis cultivation guidance based on customer's setup and experience.

## Knowledge Base
- `/knowledge/grow-guides/indoor-guide.md` — Full grow guide
- `/knowledge/products/` — Equipment catalog (SativaBox)

## Inputs
- Experience level: Beginner / Intermediate / Advanced
- Space: Tent size, outdoor, stealth
- Budget: Starter / Intermediate / Professional
- Goal: Personal use / Small yield / Maximum yield

## Process
1. Assess customer's current setup
2. Identify gaps in equipment/knowledge
3. Recommend specific products from SativaBox
4. Provide actionable growing steps
5. Warn about common beginner mistakes

## Output
- Equipment recommendation list (SativaBox links)
- Week-by-week grow schedule
- Nutrient feeding chart
- Problem prevention tips

## Safety
- Always stay within Luxembourg/EU law
- Hemp (>0.3% THC) cultivation for personal use — check current law
- Never suggest illegal activities
